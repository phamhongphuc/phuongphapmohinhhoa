# MVC Chat


## Running Locally


```sh
git clone https://github.com/phamhongphuc/phuongphapmohinhhoa # or clone your own fork
cd phuongphapmohinhhoa
npm install
npm start
```

Your app should now be running on [localhost:80](http://localhost:80/).
